# Scrapers

Python extractors that turn publicly available court data into YAML proposed for `data/`.

This directory is a stub. Expected stack: Python 3.11+, managed with [uv](https://docs.astral.sh/uv/). Tests with pytest.

Scrapers never write to production. They generate pull requests that maintainers review.

See [../docs/architecture.md](../docs/architecture.md) for how scrapers fit into the pipeline.
